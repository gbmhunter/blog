## Comments

As far as I know, the TO-Leadless package is solely used by Infineon for a range of it's high power MOSFETs. In this package, Infineon's MOSFETs can achieve on-resistances as low as 0.4mÎ©. It's package size is 60% smaller than the D2PAK 7-pin package.
