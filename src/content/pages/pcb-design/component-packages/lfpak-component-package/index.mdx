LFPAK is a family of surface-mount power packages primarily used for power MOSFETs. It was originally developed by Nexperia (the first LFPAK, the LFPAK56 was released in 2002[^wevolver-how-lfpak-revolutionized-an-industry]) as a higher performance alternative to the DPAK/D2PAK packages. Rather than internal bond wires it uses a large copper clip, which reduces package electrical and thermal resistance. It appears that the package is still exclusively used by Nexperia.

<Image src={import('./_assets/dpak-vs-lfpak-nexperia-the-history-of-lfpak-youtube.webp')} width="600px">A screenshot from a Nexperia video comparing the DPAK/D2PAK and LFPAK packages.[^nexperia-the-history-of-lfpak]</Image>

The LFPAK packages were released in the following order:

- LFPAK56: Released in 2002.
- LFPAK56E: Released in 2008.
- LFPAK33: Released in 2010.
- LFPAK56D: Released in 2011.
- LFPAK88 (SOT-1235): Released in 2019. Quoted as providing 2x higher continuous power rating compared to the [D2PAK package](/pcb-design/component-packages/to-263-component-package/).

## LFPAK33

<Image src={import('./_assets/lfpak33-3d-render.webp')} width="300px">A 3D render of the LFPAK33 (SOT-1210) package.[^nexperia-lfpak33-mosfets]</Image>

LFPAK33 is a small package in the LFPAK family, with a footprint of 3.3 x 3.3 mm. It allows for max. continuous MOSFET drain currents of about 70 A.[^nxp-power-mosfet-selection-guide] Also known as the SOT-1210.[^nxp-sot1210]

### Synonyms

- SOT-1210: Another name by Nexperia for the LFPAK33 package.

### Dimensions

| Property         | Value          |
| ---------------- | -------------- |
| Leads            | 8              |
| Pitch            | 0.65 mm        |
| Body Length (D)  | 2.6 mm         |
| Body Width (E)   | 3.3 mm         |
| Body Height      | 0.9 mm         |
| Footprint        | 3.3 x 3.3 mm   |
| Footprint Area   | 10.89 mm²      |

<Image src={import('./_assets/lfpak33-dimensions.webp')} width="600px">The dimensions of the LFPAK33 (SOT-1210) package.[^nexperia-buk7m10-40e-mosfet-ds]</Image>

### Similar To

The following packages from other manufacturers can all be soldered onto a common PCB footprint:[^nxp-power-mosfet-selection-guide]

- AOS DFN3.3x3.3
- EMC EDFN 3x3
- Fairchild MLP08S
- Fairchild PQFN08B
- Infineon TSDSON-8
- On Semi Micro8 Leadless
- Vishay PowerPak 1212-8

Diodes Inc has a similar package called the PowerDI3333. There is a published footprint by Diodes Inc (shown below) that is compatible with both the LFPAK33 and PowerDI3333 packages.[^diodes-inc-lfpak33-powerdi3333-compatible-footprint] It may also be compatible with the other packages listed above.

## LFPAK56

The LFPAK56 package was the first package released in the LFPAK family.[^wevolver-how-lfpak-revolutionized-an-industry]. It also called the SOT-669.[^nxp-sot669] It is commonly classed in a group of similar packages called Power-SO8. This includes other packages such as the Vishay PowerPAK SO8 and Infineon Super SO8 (see the Similar To section for more details). It allows for max. continuous MOSFET drain currents of about 100 A.[^nxp-power-mosfet-selection-guide]

<Image src={import('./_assets/lfpak56-3d-render.webp')} width="300px">A 3D render of the LFPAK56 (Power-SO8) package.</Image>

The LFPAK56 package offers significant PCB space savings compared to the DPAK/D2PAK packages, as shown in <IRef iref="fig-dpak-d2pak-vs-lfpak56-nexperia" />.

### Similar To

The LFPAK56 is similar to the packages listed below. All of the below packages can be soldered onto a common PCB footprint.[^nxp-power-mosfet-selection-guide]

- [Vishay PowerPAK SO8](/pcb-design/component-packages/powerpak-so8-component-package/): The PowerPAK SO8 is a Vishay package that is commonly classed as "footprint compatible" with the LFPAK56 package.

## LFPAK56E

LFPAK56E is an enhanced version of the LFPAK56 package, with an optimized lead frame and package design which results in an improved $R_{DS(on)}$ and power density improvement of up to 30%.[^eepower-superjunction-fets-in-lfpak56e-packages]
