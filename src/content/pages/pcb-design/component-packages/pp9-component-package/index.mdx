## PP9

The PP9 package is a less common 9V battery size (compared to the [PP3](/pcb-design/component-packages/pp3-component-package/)). Zinc carbon 9V batteries come in this package.

<Image src={import('./_assets/pp9-component-package-9v-eveready-silver-battery.jpg')} width="279px">A 9V battery in the PP9 component package.</Image>

Dimensions: 81(h)x66(w)x52(d)mm
