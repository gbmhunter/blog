SOT-227 is a high-power package with M4 screw terminals. It is used by companies such as Vishay and IXYS for diodes (an example rating is a 1200V, 140A diode). In many cases, all four electrical terminals are isolated from the backside (which is designed to be bonded to a heatsink).

<Image src={import('./_assets/sot-227-photo.webp')} width="300px">A photo of the SOT-227 component package.[^digikey-skis300-200-product-page]</Image>

## Synonyms

- `ISOTOP`: Microsemi's package name.[^microsemi-mounting-considerations-for-the-sot-227-package]
- `SOT-227-4, miniBLOC`: DigiKey's package name.
- `minibloc`: IXYS's package name.[^ixys-dsep2x91-03a-diode-ds]
- `SOT-227B`: Package name used by IXYS and Littelfuse (the `B` might be referring to either "backside", "bonded" or perhaps the second revision of the package, just like Vishay's "Generation 2" dimensions).[^ixys-dsep2x91-03a-diode-ds] [^littelfuse-handling-and-mounting-minibloc-sot227b]

## Thermal Resistance

To pick on a specific example in the SOT-227 package, the Vishay VS-HFA140FA120 HEXFRED 140A ultrafast soft recovery diode has a junction-to-case thermal resistance of $R_{\theta JC} = 0.35^{\circ}C/W$ (max.) when one of the legs is conducting, and $R_{\theta JC} = 0.175^{\circ}C/W$ (max.) when both legs are conducting. It has a very low case-to-heatsink thermal resistance of $R_{\theta CS} = 0.05^{\circ}C/W$ (typ.).[^vishay-vs-hfa140fa120-hexfred-140a-ultrafast-soft-recovery-diode-ds]

Microsemi's application note [Mounting Considerations for the SOT-227 Package](https://ww1.microchip.com/downloads/aemDocuments/documents/PPM/ApplicationNotes/ApplicationNotes/1818_a.pdf) provides some in-depth information on the thermal performance of the SOT-227 package.[^microsemi-mounting-considerations-for-the-sot-227-package]
