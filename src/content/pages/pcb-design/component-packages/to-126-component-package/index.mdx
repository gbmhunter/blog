TO-126 (JEDEC: Transistor Outline Package, Case Style 126) is a 3 lead through-hole component package commonly used for transistors. The package is rectangular with a hole in the middle for bonding to a heatsink. It is most easily recognized apart from similar packages by the three small indentations on the circumference of the central hole.

## Synonyms

- 12A3: The DIN name as per DIN 41869.[^wikipedia-to-216]
- A56: The IEC name as per IEC 60191.[^wikipedia-to-216]
- SOT-32: Name used by onsemi.[^onsemi-bd135-npn-transistor-ds]
- TO-225AA: The name in the updated JEDEC outline system.[^wikipedia-to-216]
