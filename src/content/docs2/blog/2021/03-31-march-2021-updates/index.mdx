Added info on overriding the standard library version of `malloc()` (and friends) with an application specific version:

```c

int main()
{
    inti = malloc(sizeof(int));
    free(i);   
}

voidmalloc(size_t s)
{
    printf("My custom malloc() called");
    return NULL;
}

void free(voidp)
{
    printf("My custom free() called");
}
```
