## Overview

import to_pmod_7_component_package_3d_render from './_assets/to-pmod-7-component-package-3d-render.jpg'

<Image src={to_pmod_7_component_package_3d_render} width="200px">A 3D render of the TO-PMOD-7 component package.</Image>

Transistor Outline Power Module 7 is a rather large SMD package with good heat sinking abilities. Texas Instruments uses this package for some of their DC/DC power modules, which have built in shielded inductors.

## Synonyms

* NDW: Used by Texas Instruments[^ti-lmz12001ext-smps-ds].
* TZA07A: Used by Texas Instruments[^ti-lmz12001ext-smps-ds].

## Dimensions And Recommended Land Pattern

import to_pmod_7_component_package_dimensions_and_land_pattern from './_assets/to-pmod-7-component-package-dimensions-and-land-pattern.png'

<Image src={to_pmod_7_component_package_dimensions_and_land_pattern} width="800px">The dimensions and recommended land pattern for the TO-PMOD-7 component package. Image from http://www.ti.com/.</Image>

Pitch is 1.27mm (between the six pins, excluding the pad).

Height: 4.57mm.

Land Area: $10.16x13.77mm = 140mm^2$

## Thermal Resistances

* $T_{JC} = 1.9^{\circ}{\rm C}/W$
* $T_{JA} = 16^{\circ}{\rm C}/W$ (with the recommended land pattern)
