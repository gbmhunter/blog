## Overview

SOD-110 (Small-outline Diode 110) is a small 2-lead SMD component package used for diodes.

import sod_110_component_package_photo from './_assets/sod-110-component-package-photo.png'

<Image src={sod_110_component_package_photo} width="357px">A photo of the SOD-110 component package.</Image>

## Synonyms

* SOD2: NXP's "package type descriptive code"[^nxp-sod110].

## Dimensions

SOD-110 is 2.0mm long, 1.25mm wide and 1.5mm high[^nxp-sod110].
