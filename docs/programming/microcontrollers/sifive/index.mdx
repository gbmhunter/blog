Microcontrollers use the open-source RISC-V instruction set.

## HiFive1 Rev B 

The _HiFive1 Rev B_ is SiFive's second revision of their first development kit. It carries a FE310-G002 _Freedom Everywhere_ SoC (System on Chip).

<Image src={require('./sifive-hifive-rev-b-dev-kit-stock-photo.png').default} width="500px">Photo of the HiFive Rev B development kit. Image from https://www.crowdsupply.com/sifive/hifive1-rev-b, retrieved 2020-12-02.</Image>

### Specifications

Parameter       | Value
----------------|-----------------------------
Input Voltage   | USB 5V or 7-12V via DC jack
IO Voltage      | 3.3V
Flash Memory    | 32Mbit (external via SPI)
Clock Speed     | 320MHz
Digital IO Pins | 19
PWM Pins        | 9
UART            | 2
I2C             | 1
