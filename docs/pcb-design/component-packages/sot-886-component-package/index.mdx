## Overview

<Image src={require('./_assets/sot-886-component-package-3d-render.jpg').default} width="200px">A 3D render of the SOT-886 component package.</Image>

SOT-886 (Small Outline Transistor 886) is a SMD component package with 6 leads. It is commonly used for RFID tag chips, logic gates, buffer/line drivers and linear voltage regulators.

NXP seems to be the only manufacturer who uses this package.

## Synonyms

* 6-XSON: DigiKey[^digikey-tvs-search].
* XSON6: NXP's name[^nxp-sot-886].

## Dimensions

Below are the dimensions for the SOT-886 component package from NXP[^nxp-sot-886]:

<Image src={require('./_assets/sot-886-package-dimensions-nxp.png').default} width="800px">The dimensions of the SOT-886 component package from NXP[^nxp-sot-886].</Image>

|------------------------|------------------|--------|
| Seated height          | $A$            | 0.5mm  |
| Body length            | $D$            | 1.45mm |
| Body width             | $E$            | 1.00mm |
| Pitch                  | $e$            | 0.6mm  |

## Recommended Land Pattern

<Image src={require('./_assets/component-footprint-sot-886-land-pattern.png').default} width="800px">The PCB land pattern for the SOT-886 component package from NXP[^nxp-sot-886].</Image>

The total occupied area of the component on the PCB is $1.70mm \times 1.25mm = 2.15mm^2$.
